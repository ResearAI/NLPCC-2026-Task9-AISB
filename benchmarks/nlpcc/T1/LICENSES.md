# Licenses And Data Access

This package contains benchmark metadata and evaluation scaffolding. Some T1 data sources may be gated or subject to their original licenses and access terms.

- HLE / HLE-Verified: download or use only under the upstream dataset terms.
- FeatureBench: follow the upstream benchmark license and repository terms.
- AISB/NLPCC scaffolding: use under this repository's license notice.

Secrets such as `HF_TOKEN` must be provided by environment variable and must not be written into submission files.
